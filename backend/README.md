# Backend

FastAPI backend for the fullstack template.

## Contents

- `backend/`: Source code for the application.
- `tests/`: Pytest tests.
- `pyproject.toml`: Configuration for dependencies and tools.

## Commands

- Run server: `uv run uvicorn backend.main:app --reload`
- Run tests: `uv run pytest`
- Format: `uv run ruff format .`
- Lint: `uv run ruff check .`
